Key terms (Terms to know)

Cookie
Small files of information of that a web server generates and sends to a web browser. Cookies help inform websites about the user, enabling the websites to personalize the user experience Source: https://www.cloudflare.com/learning/privacy/what-are-cookies/
Third party cookie
Cookie that belongs to a domain other than the one displayed in the browser. Third-party cookies are most often used for tracking purposes
Hypertext transfer protocol secure(HTTPS)
The secure version of HTTP, and primary protocol used to send data between a web browser and a website.https://www.cloudflare.com/learning/ssl/what-is-https/
Hypertext Transfer Protocol (HTTP)
foundation of the World Wide Web, and is used to load webpages using hypertext links. HTTP is an application layer protocol designed to transfer information between networked devices and runs on top of other layers of the network protocol stack
https://www.cloudflare.com/learning/ddos/glossary/hypertext-transfer-protocol-http/
Do Not Track Setting
Browser setting that sends a ‘Do not track’ request to websites when a user visits the website through requesting not to store cookies during searching the imternet. However enabling this setting is not effective on its own. https://www.cookieyes.com/knowledge-base/cookies-101/what-is-do-not-track/#:~:text=Do%20Not%20Track%20is%20a,stop%20websites%20from%20tracking%20users.
